# 📂 File Organizer Automation

## 📌 Overview
This project automatically organizes files into categorized folders based on their file extensions.

## 🛠 Technologies
- Python
- os module
- shutil module

## ▶ How to Use
1. Create a folder called `organize_here`
2. Place files inside it
3. Run `organizer.py`
4. Files will be sorted automatically

## 🎯 Purpose
Demonstrates automation, file handling, and clean code structure.
