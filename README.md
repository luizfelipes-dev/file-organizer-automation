# 📂 File Organizer Automation

## 📌 Overview
This project automatically organizes files into categorized folders based on their file extensions.  
It prevents file overwriting and handles unknown file types by placing them into an `Others` folder.

---

## 🛠 Technologies Used
- Python
- os module
- shutil module

---

## 🚀 Features
- Automatic file organization
- Prevents file overwrite (renames duplicates automatically)
- Creates categorized folders dynamically
- Handles unknown file types with an `Others` folder
- Clean and structured code

---

## ▶ How to Use

1. Clone or download the repository
2. Create a folder named `organize_here` in the project directory
3. Place the files you want to organize inside this folder
4. Run:

```bash
python organizer.py
